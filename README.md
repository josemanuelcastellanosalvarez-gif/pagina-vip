# pagina-vip

Pagina web romantica para Pequena Jess.

## Como verla

Abre `index.html` en tu navegador o ejecuta:

```powershell
start index.html
```

## Como publicarla para compartir

La opcion mas facil es GitHub Pages.

1. Crea una cuenta en GitHub si no tienes una.
2. Crea un repositorio nuevo, por ejemplo `pagina-vip`.
3. Sube todos los archivos de esta carpeta al repositorio.
4. En GitHub entra a `Settings` > `Pages`.
5. En `Build and deployment`, elige:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Guarda los cambios.
7. GitHub te dara un enlace publico para compartir.

## Otra opcion facil

Tambien puedes subirla a Netlify o Vercel arrastrando esta carpeta completa.

## Antes de compartir

- Agrega mas mp3 dentro de la carpeta `musica`
- Agrega mas fotos dentro de la carpeta `images`
- Cambia textos si quieres algo aun mas personal
- Recarga con `Ctrl + F5` para ver siempre la ultima version
